Style Guide

Please adhere to the following visual interface guidelines when working with Trader Joe’s online brand in order to establish and preserve brand integrity of Trader Joe’s online appearance.

Colors

Dark Brown: $color-dark

Hex Code: #462D23

darken($color-dark, 10%), darken($color-dark, 5%), $color-dark

Bright Red: $primary-color-1

Hex Code: #D60D34

darken($primary-color-1, 10%), $primary-color-1

Yellow: $primary-color-2

Hex Code: #FBF943

darken($primary-color-2, 10%), $primary-color-2, lighten($primary-color-2, 20%)

Bright Light Blue: $primary-color-3

Hex Code: #008D9A

darken($primary-color-3, 10%), $primary-color-3, lighten($primary-color-3, 10%), lighten($primary-color-3, 50%)

Typography

The typefaces used on this site are Hooligan and Freight Micro Pro. Please use the sizes, styles and weights accordingly.Any color except for $primary-color-2 (yellow) may be used. Note: font-size is set to 62.5% on the site, resulting in 1em = 10px.

h1: Hooligan 6.5ems (65px), 1.3 line-height.

h4: Hooligan. 5.3ems (53px), 1.6 line-height.

h2: Hooligan. 4ems (40px), 1.2 line-height.

h3: Hooligan. 3ems (30px), 1.6 line-height.

h5: Freight Micro Pro Book, 1.6ems (16px), 1.6 line-height.

Paragraphs in Freight Micro Pro Book, 1.9ems (19px), line-height 1.6. Freight Micro Pro Light may be used as well.

Blockquotes in Freight Micro Pro Book, 27px-35px. Blockquotes may be semi-bold or italic.

Website Elements

Buttons

Buttons in the global version of the site are full-width colored bars with padding top and bottom of at least 25 pixels.The text uses paragraph font styled with to text-transform: uppercase and letter-spacing 0.1em.

Find a Store

Buttons in larger versions of the site uses the same font settings but with a transparent background and 3px border, with a border radius of 5px. Padding between 10-20px for all sides should be used to set the width and height.

Find a Store

The hover state of buttons are 10% darker than it original color using a 200ms ease transition.

Dividers

Dividers in the site are top (or bottom) borders of the element below (or above). They are 3px, dotted, and any color may be used.

Iconography

A set of 10 icons are used on Trader Joe’s site, nine representing the private labels of Trader Joe’s, and one icon used in the store locator section.

Photos and Textures

All photos are copyrighted to Trader Joe’s. The textures are created in Adobe Photoshop using a Flower Brush set that can be found here. The flower texture is used on dark backgrounds, using the same color as the background with a 30% or lower opacity.
